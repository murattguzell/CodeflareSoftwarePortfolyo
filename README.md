# Codeflare Software - Portfolio

Murat GÜZEL'in kişisel portföy web sitesi. Android Developer olarak geliştirdiğim projeleri ve deneyimlerimi sergilediğim modern, animasyonlu bir web sitesi.

## Özellikler

- 🎨 Modern ve animasyonlu tasarım
- 🌍 Türkçe ve İngilizce dil desteği
- 📱 Responsive tasarım (mobil uyumlu)
- ⚡ Hızlı yükleme (statik HTML/CSS)
- 🎯 Proje detay sayfaları
- 📧 İletişim bölümü
- 🔒 Gizlilik politikası ve kullanıcı sözleşmesi

## Sayfalar

- **Ana Sayfa** - Biyografi ve hızlı erişim
- **Projeler** - Geliştirdiğim projelerin listesi
- **Proje Detayları** - Proje açıklaması, özellikler, teknolojiler
- **İletişim** - GitHub, LinkedIn, E-Mail bağlantıları
- **Gizlilik Politikası** - Veri gizliliği hakkında bilgi
- **Kullanıcı Sözleşmesi** - Kullanım koşulları
- **Hesap Silme** - Veri silme talebi

## Teknolojiler

- HTML5
- CSS3 (Animasyonlar ve Gradientler)
- Vanilla JavaScript
- GitHub Pages

## Kurulum

1. Repository'yi klonlayın
2. `index.html` dosyasını tarayıcıda açın
3. Veya GitHub Pages üzerinden erişin

## Özelleştirme

### Profil Resmi
`profile.jpg` dosyasını kendi resminizle değiştirin.

### Proje Resimleri
`project1.jpg`, `project2.jpg`, `project3.jpg` dosyalarını kendi proje resimlerinizle değiştirin.

### İletişim Bilgileri
`script.js` dosyasında GitHub, LinkedIn ve E-Mail linklerini güncelleyin.

### Proje Bilgileri
`script.js` dosyasında `projects` array'ini kendi projelerinizle güncelleyin.

## GitHub Pages Yayınlama

1. Repository'yi GitHub'a push edin
2. Repository ayarlarında "Pages" bölümüne gidin
3. Source olarak "main" branch'ini seçin
4. Custom domain eklemek için CNAME dosyası oluşturun
